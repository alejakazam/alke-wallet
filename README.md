# AlkeWallet

Proyecto desarrollado por Alejandra Duarte para el BootCamp de Talento Digital.

Aplicación web que simula una billetera virtual sin uso de backend.
Permite visualizar un saldo actual, realizar transacciones y consultar un historial simulado utilizando variables locales.

---

## I. Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap (CDN)
* JavaScript
* SweetAlert

---

## II. Descripción del sistema

El sistema permite:

* Visualizar saldo disponible
* Simular transacciones
* Mostrar historial de movimientos
* Interacción dinámica mediante JavaScript
* Uso de alertas personalizadas con SweetAlert

---

## III. Características principales

* Interfaz construida con Bootstrap mediante CDN
* Manejo de datos en el navegador (sin backend)
* Simulación de lógica de una billetera virtual
* Experiencia interactiva con alertas personalizadas
