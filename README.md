Alkewallet es un proyecto desarollado por Alejandra Duarte para el BootCamp de Talento Digital, 
en estos html están implementados elementos de bootstrap en CDN, JS y SweetAlert (especializado en mostrar alertas personalizables), 
el sistema busca simular sin un backend una estructura de billetera virtual con un saldo actual,
transacciones e historiales (simulado) utilizando variables locales para guardar los valores.
